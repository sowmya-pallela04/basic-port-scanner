# Basic Port Scanner

A simple Python-based TCP port scanner.

## Features
- Checks host availability using ping
- Scans common ports (22, 80, 443)
- Uses Python socket programming

## Technologies Used
- Python
- Socket module
- Networking basics
# NetWatch

NetWatch is a simple Python network monitoring tool.

## Features
- Checks network connectivity
- Displays connection status
- Lightweight and easy to use

## Requirements
- Python 3.7
